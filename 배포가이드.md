## 🚀 방법 1: Vercel 배포 (5분 완성!)

### **준비물**
- GitHub 계정 (또는 Google 계정)
- 다운로드한 `index.html` 파일

### **Step 1: Vercel 가입**

1. https://vercel.com 접속
2. "Sign Up" 클릭
3. **"Continue with GitHub"** 선택 (가장 쉬움)
   - 또는 "Continue with Email" 선택
4. GitHub 로그인하면 자동 연동!

### **Step 2: 프로젝트 배포**

#### **방법 A: 드래그 & 드롭 (초간단!)**

1. Vercel 로그인 후 대시보드
2. **"Add New..."** → **"Project"** 클릭
3. **"Deploy"** 탭 선택
4. **드래그 앤 드롭 영역**에 `index.html` 파일 끌어다 놓기
5. 자동으로 배포 시작! ⚡
6. 완료되면 **"Visit"** 버튼 클릭
7. 🎉 **완성!** 이제 인터넷 어디서나 접속 가능!

#### **방법 B: GitHub 연동 (추천!)**

1. GitHub에 새 저장소 만들기:
   ```
   이름: crypto-autotrader
   Public 선택
   "Create repository" 클릭
   ```

2. 파일 업로드:
   ```bash
   # 터미널에서
   cd (index.html이 있는 폴더)
   
   git init
   git add .
   git commit -m "첫 배포"
   git branch -M main
   git remote add origin https://github.com/당신의아이디/crypto-autotrader.git
   git push -u origin main
   ```

3. Vercel에서 Import:
   - Vercel 대시보드
   - "Add New..." → "Project"
   - "Import Git Repository" 
   - GitHub 저장소 선택
   - "Deploy" 클릭!

### **Step 3: 확인**

배포가 완료되면:
```
✅ 자동 URL: https://당신의프로젝트.vercel.app
✅ HTTPS 보안 연결
✅ 모바일 자동 지원
✅ 전세계 CDN (빠른 속도)
```

### **Step 4: 휴대폰에서 테스트**

1. 휴대폰 브라우저 열기
2. Vercel URL 입력
3. 홈 화면에 추가:
   - **iPhone**: Safari → 공유 → "홈 화면에 추가"
   - **Android**: Chrome → 메뉴 → "홈 화면에 추가"
4. 이제 앱처럼 사용! 📱

**결과:** `https://랜덤이름.netlify.app`

## 🚀 방법 3: GitHub Pages (무료 호스팅)

### **Step 1: GitHub 저장소 준비**

```bash
cd (index.html이 있는 폴더)

git init
git add .
git commit -m "GitHub Pages 배포"
git branch -M main
git remote add origin https://github.com/당신의아이디/crypto-autotrader.git
git push -u origin main
```

### **Step 2: GitHub Pages 활성화**

1. GitHub 저장소 페이지
2. **Settings** 탭
3. 왼쪽 메뉴에서 **Pages** 클릭
4. **Source**: "Deploy from a branch" 선택
5. **Branch**: "main" 선택
6. 폴더: "/ (root)" 선택
7. **Save** 클릭

### **Step 3: 확인**

5분 후:
```
✅ URL: https://당신의아이디.github.io/crypto-autotrader
```

## 🎨 나만의 도메인 연결 (선택사항)

### **무료 도메인 받기**

**Freenom** (무료):
1. https://freenom.com
2. 원하는 도메인 검색
3. `.tk`, `.ml`, `.ga` 등 무료 선택
4. 등록 (이메일 필요)

**유료 도메인** (추천):
- Namecheap: $8/년
- GoDaddy: $10/년
- Gabia (한국): 1만원/년

### **Vercel에 도메인 연결**

1. Vercel 프로젝트 → **Settings**
2. **Domains** 탭
3. "Add" 클릭
4. 도메인 입력 (예: mytrader.com)
5. Vercel이 알려주는 DNS 설정 복사
6. 도메인 제공업체에서 DNS 설정
7. 완료! ✅

## 🆘 문제 해결

→ 저장소 루트에 파일 있는지 확인

→ 인터넷 연결 확인
