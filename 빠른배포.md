# ⚡ 1분만에 인터넷 배포하기!

## 🎯 가장 빠른 방법 (드래그앤드롭)

### **Step 1: Vercel 가입** (30초)
1. https://vercel.com 접속
2. "Sign Up" 클릭
3. "Continue with GitHub" 선택
4. GitHub 로그인

### **Step 2: 파일 업로드** (30초)
1. Vercel 대시보드에서 "Add New..." 클릭
2. "Project" 선택
3. **"Deploy"** 탭 클릭
4. 이 폴더의 **모든 파일**을 드래그해서 드롭!
   - index.html
   - vercel.json
   - README.md
5. 자동 배포 시작! ⚡

### **Step 3: 완료!** (자동)
- 배포 완료되면 **"Visit"** 버튼 클릭
- 🎉 **인터넷 주소 복사!**
- 예: `https://crypto-autotrader-abc123.vercel.app`

---

## 📱 휴대폰에서 접속하기

1. 휴대폰 브라우저 열기
2. 위에서 받은 주소 입력
3. 완벽하게 작동! ✅

### 앱처럼 사용하기:

**iPhone:**
- Safari → 공유(↑) → "홈 화면에 추가"

**Android:**
- Chrome → 메뉴(⋮) → "홈 화면에 추가"
