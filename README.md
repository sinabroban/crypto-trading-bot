# 🚀 Crypto AutoTrader Pro

암호화폐 자동매매 시뮬레이터 - 인터넷 어디서나 접속 가능!

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

## 🌐 온라인 데모

**지금 바로 사용해보세요!**

👉 [https://crypto-autotrader.vercel.app](배포 후 여기에 실제 URL 입력)

## ✨ 주요 기능

- 📊 **실시간 시세 모니터링** - BTC, ETH, XRP
- 🤖 **자동매매 시뮬레이션** - 5가지 전략
- 📱 **완전 반응형** - PC, 태블릿, 모바일 완벽 지원
- 🎯 **손절/익절 자동 설정**
- 📈 **실시간 차트**
- 💰 **가상 자산 관리**

## 🎮 사용 방법

### 1️⃣ 웹사이트 접속
- PC: 브라우저에서 URL 입력
- 모바일: 브라우저에서 접속 후 홈 화면에 추가

### 2️⃣ 전략 설정
1. **전략 설정** 탭 클릭
2. 원하는 전략 선택
3. K값, 손절/익절 설정

### 3️⃣ 시뮬레이션 시작
1. **시작** 버튼 클릭
2. 실시간으로 거래 확인
3. 로그에서 상세 내역 확인

## 📱 모바일 앱처럼 사용하기

### iPhone
1. Safari에서 사이트 접속
2. 공유 버튼 (↑) 클릭
3. "홈 화면에 추가" 선택
4. 앱 아이콘으로 사용!

### Android
1. Chrome에서 사이트 접속
2. 메뉴 (⋮) 클릭
3. "홈 화면에 추가" 선택
4. 앱 아이콘으로 사용!
